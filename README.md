# task-board

React で作成したタスク管理アプリケーションです。

## 機能

- タスクの追加（テキスト入力後に「追加」ボタンまたは Enter キー）
- チェックボックスで完了・未完了の切り替え
- 削除ボタン（✕）でタスクを除去
- 完了済みタスクはグレー表示 + 取り消し線
- 完了件数カウンター表示
- localStorage によるデータ永続化（リロードしてもタスクが消えない）

## 技術スタック

- React 18
- Vite 5

## セットアップ

```bash
npm install
npm run dev
```

http://localhost:5173 でアクセスできます。

## Claude Code での開発について

このリポジトリは Claude Code を使って開発を進めています。

### 事前準備

GitHub CLI をインストールし、以下のコマンドで認証しておくことで、Claude がコミット・プッシュから Pull Request の作成・マージまで一貫して完全自動で行えます。

```bash
gh auth login
```

認証後は Claude に指示するだけで、ブランチ作成 → コミット → プッシュ → PR 作成 → マージまで自動で対応します。
